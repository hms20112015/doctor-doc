# doctor-doc
This repo uses llama2 to generate documented functions based on user input.

# Quickstart

1. Ensure you have the `llama2` model installed:

   ```bash
   ollama pull llama2
   ```

2. Install the Python Requirements.

   ```bash
   pip install -r requirements.txt
   ```

3. Run the example:

   ```bash
   python doctordoc.py
   ```
